# Islands Theme for Zed

A theme for [Zed](https://zed.dev) inspired by JetBrains' Islands design system, with both dark and light variants.

## Installation

1. Open Zed
2. Go to **Extensions** (⌘⇧X)
3. Search for "Islands Theme"
4. Click **Install**

## Colors

Based on the [Islands color palette](https://github.com/himattm/iterm2-islands), following JetBrains' syntax highlighting conventions:

- Keywords in blue
- Strings in green
- Functions in yellow
- Types in cyan
- Constants in magenta
